Sim.I.am

A MATLAB-based educational bridge between theory and practice in robotics.

Download .zipUnzip and launch in MATLAB 2011a or later. Get .mlappinstallInstall and run as an app in MATLAB 2012b. View on GitHubSee the source code.
Requires MATLAB. Tested on Linux and Windows, MATLAB 2012a and 2012b.

Instructions

If you are using MATLAB 2012b, download the .mlappinstall file.
  1. Double click-on simiam_demo.mlappinstall from inside of MATLAB.
  2. Press 'Install' to install this application into 'My Apps'.
  3. Select the 'APPS' tab, click on the drop-down arrow, and select 'simiam_demo' from 'My Apps' to start the application.
If you are using MATLAB 2011a or later, download the .zip file.
  1. Extract simiam_demo.zip to a location of your choice.
  2. Navigate inside of MATLAB to where the .zip was extracted and enter the directory simiam_demo.
  3. Use the command launch to start the application.
